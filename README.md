# symCAP Symbiosis Custom Annotation Pipeline
Script used to annotate genomes from primary symbionts in planthoppers

# Required software

[HMMER](http://hmmer.org/download.html)

[rnammer](https://vcru.wisc.edu/simonlab/bioinformatics/programs/install/rnammer.htm)

[tRNAscan-SE](http://lowelab.ucsc.edu/tRNAscan-SE/)

[mafft](https://mafft.cbrc.jp/alignment/software/)

[align_nucl_by_codon2.py](https://github.com/diecasfranco/symCAP/blob/master/align_nucl_by_codon2.py)
